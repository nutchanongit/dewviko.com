## 2. Messaging & Pain Points ที่คู่แข่งใช้

### Pain Points ที่ถูกพูดถึงบ่อย:
1. **"ทำวิดีโอไม่เป็น แต่ต้องลงคอนเทนต์"** — เจ้าของธุรกิจที่ไม่มีทีมวิดีโอ
2. **"จ้างคนตัดต่อแพง"** — ค่าใช้จ่ายผลิตคลิป 1 ชิ้น vs. ระบบ AI ทำเอง
3. **"ไม่รู้จะเริ่ม AI ยังไง"** — คนอยากใช้ AI แต่เครื่องมือเยอะจนสับสน
4. **"ต้องเขียนโค้ดใช่ไหม?"** — กลัว technical barrier
5. **"ลงคอนเทนต์ไม่ทัน"** — ต้องผลิตคลิปเยอะๆ สม่ำเสมอ

### Value Propositions ที่ใช้:
- "ไม่ต้องเขียนโค้ด" (No-code / Low-code)
- "สอนแบบจับมือทำ step-by-step"
- "ใช้ได้จริงกับธุรกิจ"
- "เรียนจบทำได้เลย"
- "สร้างระบบอัตโนมัติ ลดงานซ้ำซ้อน"

## 3. Creative Patterns ที่ทำงานได้ดีบน TikTok (จาก research)

### Pattern A: "ก่อน vs หลัง" (Before/After)
- แสดงการทำวิดีโอแบบเดิม (นั่งตัดต่อ 3 ชม.) → กดปุ่มเดียวได้คลิปจาก AI
- **ทำไมได้ผล:** เห็นความแตกต่างชัด, กระตุ้น FOMO

### Pattern B: "สาธิตสด" (Live Demo / Screen Recording)
- อัดหน้าจอ n8n workflow ทำงานจริง → output เป็นคลิปสำเร็จรูป
- **ทำไมได้ผล:** พิสูจน์ว่าจริง ไม่ใช่แค่พูด

### Pattern C: "ตัวเลขน่าตกใจ" (Shocking Number Hook)
- "ผมผลิตคลิป 30 ชิ้นใน 1 วัน โดยไม่ต้องเปิดกล้อง"
- **ทำไมได้ผล:** สร้าง curiosity gap, หยุด scroll

### Pattern D: "ถามคำถาม" (Question Hook)
- "ยังนั่งตัดต่อวิดีโอเองอยู่เหรอ?"
- **ทำไมได้ผล:** คนที่มี pain point จะหยุดดู

### Pattern E: "UGC / Talking Head"
- พูดตรงกับกล้อง แบบเล่าให้เพื่อนฟัง ไม่ polish
- **ทำไมได้ผล:** TikTok ชอบ authentic content, ดู trustworthy

## 4. แนะนำ TikTok Ad Strategy สำหรับ Ebook 789 บาท

### 4.1 Ad Format ที่ควรใช้
**In-Feed Ads** (Spark Ads ถ้าเป็นไปได้) — เพราะ:
- ดูเหมือน organic content → engagement สูงกว่า
- ราคาถูกกว่า format อื่น
- เหมาะกับ digital product ราคาต่ำ-กลาง

| **เจ้าของธุรกิจ SME** | 25-45 | E-commerce, ขายของออนไลน์ | ต้องผลิตคลิปรีวิวสินค้าเยอะ แต่ไม่มีทีม |

## 7. สรุป Action Items

1. **สร้าง TikTok Ads account** + ติดตั้ง TikTok Pixel บน landing page
2. **ถ่าย ad creative 3-5 ตัว** ตาม script ด้านบน (เน้น Script 1 + 4 ก่อน)
3. **ตั้ง campaign** ด้วย settings ตาม Section 4.4
4. **รันทดสอบ 7 วัน** → ดู CTR และ CPC
5. **Kill creative ที่ CTR ต่ำกว่า 1%** → เพิ่มงบให้ตัวที่ CTR สูง
6. **สัปดาห์ 2**: สร้าง Retarget campaign สำหรับคนดูวิดีโอ 75%+
7. **วัด ROAS**: ถ้า cost per purchase < 200 บาท = กำไร (789 - ต้นทุน)

## 8. Key Metrics ที่ต้อง track

| Metric | เป้าหมาย | ทำไมสำคัญ |
|--------|----------|----------|
| **CTR (Click-Through Rate)** | > 1% | วัดว่า creative ดึงดูดพอไหม |
| **CPC (Cost Per Click)** | < 5 บาท | ควบคุมต้นทุนต่อคลิก |
| **CVR (Conversion Rate)** | > 2% | วัดว่า landing page ขายได้ไหม |
| **CPA (Cost Per Acquisition)** | < 200 บาท | ต้นทุนต่อการขาย (789 - 200 = กำไร 589) |
| **ROAS** | > 3x | คืนทุน 3 เท่า = ยั่งยืน |
| **Video Watch Rate** | > 25% | วัดว่า hook ดีพอไหม |
