MCP คือ อะไร?

พูดง่ายๆ MCP คือ สายเชื่อมต่อ

ที่ให้ AI ไปใช้งานเครื่องมือภายนอกได้

อยากให้ AI ใช้ Notion ก็ต่อ Notion MCP

อยากให้ AI ใช้ n8n ก็ต่อ n8n MCP

อยากให้ AI เปิดเบราว์เซอร์ ก็ต่อ Browser MCP

สำหรับ use case นี้ ต้องต่อ Browser MCP

ซึ่งตัวยอดนิยมที่คนใช้กันเยอะมีประมาณนี้

Puppeteer MCP

Playwright MCP

อันไหนก็ได้ แต่ Feature อาจต่างกันนิดหน่อยครับ

วิธีติดตั้ง Puppeteer MCP

claude mcp add puppeteer -- npx -y @modelcontextprotocol/server-puppeteer

แค่บรรทัดเดียว เสร็จ

อยากเช็คว่าติดตั้งสำเร็จไหม

เปิด Claude Code แล้วพิมพ์ /mcp

ถ้าขึ้น puppeteer connected = พร้อมใช้งาน

อย่าลืมติดตั้ง Node.Js ด้วยนะครับ บางเครื่องอาจจะยังไม่มี

พิมพ์เช็ค ใน Command Line แบบนี้ครับ

node --version

ถ้ามันตอบเลขเวอร์ชั่นมาแสดงว่ามีแล้ว

(Node.Js คือเปรียบเสมือนเครื่องยนต์ ที่ใช้ทำงานคู่กับ puppeteer)

  • สรุปออกมาเป็นอะไร (ตารางเปรียบเทียบ + คอมเม้นต์)
  • บันทึกไว้ที่ไหน (ที่่ต้องการบันทึก อาจจะเป้น Update-Pricing-folder)
