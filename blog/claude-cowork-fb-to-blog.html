ให้ Claude อัพเดทบทความจากเฟสบุ๊คลงเว็บไซต์อัตโนมัติ

โพสบทความในเฟส แต่ขี้เกียจอัพลงเว็บไซต์ตัวเอง? สร้าง Skill + Cronjob ให้ Claude Cowork จัดการให้ทั้ง 7 ขั้นตอน เหลือ 0

Workflow การทำงานปกติของผม

เขียนคอนเท้น + ทำภาพ ลงเฟสบุค จากนั้นนำบทความที่เขียน กับ รูป ไปเซฟลงคอม แล้วนำไปจัดเรียงให้เป็น HTML Page สำหรับบทความ สุดท้ายกด Commit/Push บทความใหม่ลง Github เพื่อส่งไปหน้าเว็บ

จะเห็นใช่ไหมครับ ว่ามันยุ่งยากมากเลย

และบอกตรงๆ ที่ผ่านมา workflow แบบนี้ ผมไม่ทำหรอก ขี้เกียจ ก็เลยไม่มีคอนเท้นลงเว็บไซต์ตัวเอง 5555

ซึ่งการที่ไม่มีคอนเท้นลงเว็บไซต์ตัวเอง มันมีข้อเสียดังนี้ครับ

  • AI หาเราไม่เจอ
  • ขาด Traffic หน้าเว็บไซต์ จากคนใช้ Google Search
  • เว็บร้าง เป็นป่าช้า

แต่ตอนนี้ ปัญหานี้ได้หมดไปแล้วครับ ผมสร้าง Skill ให้ Claude Cowork ช่วยเอาบทความและรูปในเพจเฟสบุคผม ที่เขียน ไปอัพเดทลงเว็บไซต์เรียบร้อย

อธิบายทีละ Step ว่า Skill นี้สั่งให้ Claude ทำอะไรบ้าง

ผมตั้ง Cron Job ให้มันทำแบบนี้ทุกๆ 9 โมงเช้า

Step 1 อ่านบทความที่มีอยู่แล้ว
สั่งให้มันไปดูก่อนว่า เว็บผมมีบทความอะไรอยู่แล้วบ้าง จะได้ไม่สร้างซ้ำ

Step 2 ไปดึงโพสจากเฟสบุค
ใช้ Apify (เครื่องมือ scrape ข้อมูล) ไปดูดโพสจากเพจผมย้อนหลัง 7 วัน

Step 3 กรองเอาแต่บทความจริง
โพสสั้นๆ แบบ "จดคับบ" มันจะข้ามไป เอาแต่โพสที่ยาวกว่า 200 ตัวอักษร คือเป็นบทความจริงๆ

Step 4 สร้างหน้า HTML
เอาบทความที่กรองได้ มาแปลงเป็นหน้าเว็บ โดยใช้ template ที่ผมทำไว้ ใส่ SEO tag, OG tag, GA4 tracking ให้ครบ

Step 5 โหลดรูปจากโพส
ใช้ Playwright (เครื่องมือควบคุม browser) ไปเปิดโพสเฟสบุค แล้ว screenshot รูปมาเซฟ

Step 6 อัพเดทหน้า Blog + Sitemap
เพิ่ม card บทความใหม่ในหน้า listing แล้วก็อัพเดท sitemap ให้ Google รู้ว่ามีหน้าใหม่

Step 7 Commit + Push ขึ้น GitHub
สั่ง git add, commit, push อัตโนมัติ พอ push ปุ๊บ GitHub Pages deploy ให้เลย เว็บอัพเดททันที

ทั้งหมดนี้ ผมไม่ต้องทำเองสักขั้นตอนเดียว

ผลลัพธ์จริง

ก่อนหน้านี้ ผมเขียนบทความลงเฟสอย่างเดียว เว็บร้างเป็นป่าช้า เพราะขี้เกียจทำ workflow ใน n8n ทั้งหมดเอง แค่ทำให้ลูกค้าก็เหนื่อยแล้ว

ตอนนี้ ผมแค่สั่ง Claude Cowork ทีเดียว จากที่ต้องทำ 7 ขั้นตอนเอง เหลือ 0

เว็บผมมีบทความจริงแล้วครับ จากที่ร้างมาตั้งแต่สร้างเว็บ 555

สรุปแบบดิววิโกะ

การสร้าง Skill + Cronjob ช่วยให้ Claude ทำงานที่เราต้องทำซ้ำๆ ทุกๆวัน แทนเราได้จริง

ถ้าผมต้องมาไล่ทำแบบนี้เองคงจะเสียเวลา 10-30 นาทีต่อวัน คิดเป็นเดือน ก็น่าจะประมาณ 8 ชั่วโมงต่อเดือน หรือ 1 ปีก็ 96 ชั่วโมง เอาเวลานี้ไปเรียนสกิลใหม่ๆ หรือไปพักผ่อนได้อีกเยอะ

หวังว่าบทความนี้จะมีประโยชน์ต่อเพื่อนๆ
