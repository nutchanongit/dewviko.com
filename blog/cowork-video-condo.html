Claude AI 22 มีนาคม 2026

ใช้ Claude Cowork เปลี่ยนรูปคอนโดเป็นวิดิโอ Ken Burns สวยๆ

ลองเอารูปทรัพย์จากเว็บขายคอนโดมาสร้างวิดิโอด้วย Claude Cowork + FFmpeg กะจะลองเล่นๆ แต่มันทำได้ดีเฉยเลย พร้อมสรุปว่า Cowork กับ n8n ใช้ตัวไหนเมื่อไหร่

ผมเริ่มกลัว Cowork ละ 555

ล่าสุดผมลองเอา ภาพทรัพย์จากในเว็บขายคอนโด มาสร้างเป็นวิดิโอ กะจะลองเล่นๆ แต่มันทำได้ดีเฉยเลย

วิธีการทำ

1) สร้าง Folder

  • เก็บรูปทรัพย์นั้นหลายๆ มุม
  • ไฟล์บอกรายละเอียดทรัพย์

2) เข้าไปที่ Cowork กดเลือกให้ Claude เข้าไปอ่านและทำงานได้ใน Folder นี้

3) Prompt บอก เช่น

ต้องการนำภาพจาก Folder พวกนี้ มารวมเป็น Video Ken Burns หลายๆ มุม พร้อมใส่ Subtitle บอกรายละเอียด โดยไปอ่านจากไฟล์ รายละเอียด.txt โดยใช้ FFmpeg

4) รอมันทำงาน รอรับวิดิโอ แค่นี้จบ

แล้วถ้าอยากทำระบบผลิตวิดิโออัตโนมัติล่ะ?

แต่หากใครที่ต้องการทำระบบผลิตวิดิโออัตโนมัติ ที่ไม่ใช่แค่เอาภาพมาทำ Slideshow กับแค่ใส่ซับแบบนี้

คงจะต้องใช้ n8n อยู่ เพื่อเชื่อมต่อ API ของ AI ตัวอื่นๆ สำหรับ GenVDO โดยเฉพาะ ก่อน

เช่น

  • HeyGen => Clone Human
  • ElevenLabs => Clone เสียง, สร้างเสียงพากย์
  • Kling AI => Img to VDO
  • VEO3 => Text to VDO

จากนั้นมาทำ workflow ใน n8n ให้ได้โรงงานผลิตวิดิโอตามที่ต้องการโดยเฉพาะ

สรุปแบบดิววิโกะ

Cowork น่ากลัวมาก และมันจะเก่งขึ้นเรื่อยๆ เมื่อก่อนงานแบบนี้ ผมยังใช้ n8n ทั้งหมด แต่ตอนนี้เริ่มแบ่งงานแบบนี้มาให้ Cowork บ้างแล้ว

สำหรับผม
n8n => งานที่ต้องการ workflow ชัดเจน ตรงไปตรงมา
Claude => สำหรับงานวิเคราะห์เยอะๆ ซับซ้อนๆ อยากให้คิดได้หลายตลบแก้ปัญหาเองได้

แต่ก็ไม่รู้ ผมอาจจะคิดผิดก็ได้ อนาคต Claude อาจจะมาแทน n8n ได้หมดเลย
